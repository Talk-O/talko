# Distribution Model

The TalkO SDK kit is delivered privately to customers.

## Current model

- No public npm package.
- No public PyPI package.
- No public Packagist package.
- Distribution happens through a private repository share or private archive.

## Why this matters

- Customer documentation must not point to fictional public packages.
- Example imports should reference local files inside the kit.
- Versioning is controlled by TalkO release management, not by public registries.

# Support

This kit is maintained as part of the TalkO customer integration program.

## Recommended support path

1. Reproduce the issue with the curl examples in `examples/curl/`.
2. Confirm the request against the resource map in `docs/resource-map.md`.
3. Collect the HTTP status code, response body, and relevant request headers.
4. Contact TalkO support with the failing endpoint, timestamp, and tenant context.
